# Query Vault

1. Resolve the vault and read its agent rules and root index.
2. Search titles, descriptions, tags, and filenames before opening pages. Read the smallest relevant set, then follow links only when the question requires another hop.
3. Answer from recorded knowledge and cited sources. Label synthesis, conflicts, and missing evidence clearly.
4. Cite the concept paths that support the answer.

Keep the query read-only. Finish when the answer is grounded in the vault, material conflicts or gaps are visible, and no files changed.
